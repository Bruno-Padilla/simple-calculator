# Simple Calculator

## Description

This is a simple calculator for the Odin Project Curriculum that works with addition, substraction, divition and multiplication operands.

## Used Technologies

Vanilla HTML, CSS and JS.
Keyboard support.

## How to use it

1. Start by typing a number (pressing the buttons or with the keyboard).
2. Now use an operator as `+ - x /`.
3. Type another number.
4. Since here you can keep typing operators and numbers alternating between them.
5. Once you finish go ahead an click the `=` button to evaluate the operation and see the final result.
6. You can also use the `.` for floating numbers.
7. Click `<--` to erase a digit if u commit a mistake.
8. Click `clear` instead to clear the screen and start fresh with a new operation.
